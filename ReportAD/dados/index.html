Documentação dos servicos

Estes servições são responsáveis pela comunicação com o banco de dados do sistema financeiro. A extrutura da URL é mostrada a seguir:

ipdoservidor/?metodo&tabela&argumento1&arqumento7&...

Lista de servicos

  • Gerenciar tipos
    • ip/?insert&tipo&id&nome - o id pode ser qualquer valor.
    • ip/?delete&tipo&id&nome - consulta por id, portanto o nome pode ser qualquer valor.
    • ip/?update&tipo&id&nome - informar ambos os dados
    • ip/?select&tipo&id&nome - informar no mínimo o id
    • ip/?list&tipo&id&nome - id e nome pode ser qualquer valor
